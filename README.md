
## How to Run

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/kindergarten-website.git
    ```

2. Navigate to the project folder:
    ```bash
    cd kindergarten-website
    ```

3. Open `index.html` in your web browser to view the site:
    ```bash
    open index.html
    ```
